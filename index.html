ABOUT ME

I am a friendly and enthusiastic games design graduate from Bournemouth University looking for jobs in games design, graphics and UI design, web development or other creative software industries.

UNITY EXPERIENCE

I have been developing my skills in unity since my first year at Uni and continue to use them in my own projects. The 4 examples are from all 3 years at Uni plus something I've been working on in my own time recently.

In first year we were tasked with coding the physics for a marble run in unity. The project needed to include a number of unique parts, with one breaking the laws of physics. Using C# and simple block elements, I created a complex system and added an "anthropomorphiser" which animated the marbles to walk back to the start as my physics breaking component. I was happy to recieve a 1st grade for this work.

For Interface Design in second year I needed to create a 3D gem collection game and add an user interface to it.

I used a combination of free 3D assets to create a dessert scene and matched the style of the UI with a 19th century American western theme. As part of the assignment we were required to create a point and click movement system for the player so I implemented a navmesh to achieve this.

For my dissertation, I chose to use unity as I wanted to create a mobile audio game. Instead of doing the more practical variation, I chose to do a standard project which weighted my research and dissertation higher.

"Queenie" is my current project that I'm doing for fun in my space time. It's a bee hive simulator/defense game where you can watch an army of bee's evolve.

UNREAL EXPERIENCE
